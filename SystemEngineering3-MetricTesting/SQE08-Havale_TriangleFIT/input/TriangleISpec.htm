Functional Integration Testing - Triangle

 

In this project we try to validate different possible inputs available for a valid sides of triangle. In order to calculate area and perimeter of the triangle, we have to make sure that input entered by user, basically 3 sides of triangle are valid and follow the properties of triangle.

Here we have one table with the first row as fully qualified name of the class "Triangle" on which we will be operating. The next row in table lists different sides of the triangle as the variables defined in "Triangle". And next rows down the line represent the test data that will be used. We'll use these fixtures for testing.

Some of the basic tests that are performed, negative length not allowed , zero length not allowed, sum of two smallest side of triangle should be greater than third side and so on
